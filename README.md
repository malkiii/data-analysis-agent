# Data Analysis Agent

## Local Setup

1. Clone the repository using Git or Github CLI.

2. Create a virtual environment (Optional).

3. Install the required dependencies:

   ```sh
   pip install -r requirements.txt
   ```

4. Rename ...

5. Run ... using:
   ```sh
   python app.py
   ```
